# Ders Bilgi Paketi Veritabanı

Bu proje, Karadeniz Teknik Üniversitesi'nin ders bilgi paketi sisteminin veritabanı modellemesi için hazırlanmıştır.

## Dosya Açıklamaları
- `1-veritabani-olusturma.sql`: Veritabanının oluşturulması.
- `2-tablo-olusturma.sql`: Tabloların tanımlanması ve ilişkilerin kurulması.
- `3-ornek-veri-ekleme.sql`: Tablolara örnek verilerin eklenmesi.
- `4-sql-sorgulari.sql`: Örnek sorguların çalıştırılması.
